Date Picker Editor

Click a date cell to type directly, or use the calendar button to pick from a popup.

Theme

Date Column With Columns To The Right

The date column is in the middle with other columns on both sides. Click a date cell to see the calendar button overlap.

Date As Rightmost Column

The date column is the last column. The calendar button appears below (or above) the cell since there's no room to the right.

Full-Width Table, Date At Viewport Edge

Table spans the full container. The rightmost date column is flush with the viewport edge — the button should appear below the cell.

Narrow Date Column

Date column with a fixed narrow width, surrounded by other columns.

Mixed Editors

Multiple date columns alongside other editor types.

Min / Max Constraints

Dates restricted to 2024. Disabled dates are greyed out.

Bottom of Page (Button Above)

This table is near the bottom of the page. Click the last row's date — if there's not enough room below, the button appears above the cell.
